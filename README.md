# Turn-Table-Generator URL
https://github.com/KEA240000/Turn-Table-Generator

This tool is to help you make a turn table of your completed scene/mesh and quickly export it to your desired location!
With this tool, turntables will require less fumbling with Maya's built in animation systems, and condense it down to just pressing a few buttons.

Features:
- Customizable frames per second (to allow slower turn tables).
- How many times it rotates in one video.
- Easy export to a pre-determined location!
